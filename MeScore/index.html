MeScore Analysis

Neuroplasticity & Ethical Engagement Study

Overview

This project analyzes how ethical engagement apps affect prefrontal cortex strengthening. Using Python, I process user data to calculate "ME-Scores" based on decision-making patterns.

Technical Stack:
• Language: Python (analysis.py)
• Dataset: mescore_data.csv
• Focus: Somatic marker hypothesis & neuroplasticity

The results of this analysis are used to build the ethical framework for the NeuroLiterate curriculum and the TAP Protocol.
